## Descrição

* **atividades/** → atividades desenvolvidas em sala de aula
* **exercicios/** → exercícios feitos em casa para prática e revisão
* **comandos/** → resumos e exemplos dos principais comandos SQL
* **base.sql** → estrutura base utilizada nos exemplos e exercícios
